# Ophelia Web MVC

Integration components for ASP.NET Core MVC, providing base classes and filters for a standardized development experience.

## 📂 MVC Components

### [Controllers](./Controllers)
Base controller classes that provide simplified access to services, session data, and standardized result mapping.

### [Models](./Models)
Base ViewModel types designed to work with Ophelia's UI controls and service result patterns.

### [ActionFilters](./ActionFilters)
Custom attributes to inject logic (logging, validation, error handling) across multiple controller actions.

### [Middlewares](./Middlewares)
Request pipeline components for handling localization, exception transformation, and session initialization.

### [Controls](./Controls) & [Html](./Html)
Razor-specific view components and HTML helpers for constructing complex UI elements.
