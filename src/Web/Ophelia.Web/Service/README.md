# Ophelia Web Service

Standardized request models for Web APIs built with Ophelia.

## 📂 Components

### [WebApiObjectRequest.cs](./WebApiObjectRequest.cs)
A standard wrapper for POST/PUT requests containing a single entity or object.

### [WebApiCollectionRequest.cs](./WebApiCollectionRequest.cs)
A request model designed for list endpoints, supporting filtering, sorting, and pagination parameters.
