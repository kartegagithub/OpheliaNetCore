# Ophelia Workflow & Dependency Injection

This module provides attributes and interfaces for building decoupled, injectable components within the framework.

## 📂 Components

### [InjectionPointAttribute.cs](./InjectionPointAttribute.cs)
Used to mark properties or parameters that should be resolved and injected by the framework's internal dependency resolver.

### [IInjectable.cs](./IInjectable.cs)
A marker interface for classes that are eligible for automatic discovery and injection.
