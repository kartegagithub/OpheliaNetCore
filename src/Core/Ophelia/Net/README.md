# Ophelia Net

Networking and HTTP utilities for the Ophelia Framework.

## 📂 Subdirectories

### [Http](./Http)
Focused on simplifying HTTP requests and API integrations.

- **[RequestFactory.cs](./Http/RequestFactory.cs)**: A central factory for creating and configuring `HttpRequestMessage` and `HttpClient` instances. It supports custom headers, timeouts, and payload serialization.
  - **Key Methods**: `CreateRequest`, `SendAsync`, `BuildContent`.
