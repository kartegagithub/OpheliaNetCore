# Ophelia Core Data Structures

Foundational data logging and auditing models used by the core framework.

## 📂 Components

### [SQLLog.cs](./Model/SQLLog.cs)
A model for capturing and logging raw SQL queries executed by the framework, including duration and parameters.

### [EntityLoadLog.cs](./Model/EntityLoadLog.cs)
Records the lifecycle of entity loading operations, useful for profiling data access performance.
