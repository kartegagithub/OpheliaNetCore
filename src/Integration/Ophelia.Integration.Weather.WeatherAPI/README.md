# Ophelia Integration Weather API

Integration with external weather data providers to provide real-time atmospheric insights.

## 📂 Core Services

### [WeatherService.cs](./WeatherService.cs)
Fetches current weather, forecasts, and historical data based on location coordinates or city names.
- **Key Methods**: `GetCurrentWeatherAsync`, `GetForecastAsync`.

## 📁 Subdirectories
- **[Model](./Model)**: Structured models for weather metrics (Temperature, Humidity, Wind, etc.).
