# Ophelia Integration Facebook Login

OAuth2 integration for authenticating users via Facebook.

## 📂 Core Services

### [FacebookLoginService.cs](./FacebookLoginService.cs)
Interacts with the Facebook Graph API to validate user tokens and fetch profile data.
- **Key Methods**: `GetUserDataAsync`, `ValidateToken`.

## 📁 Subdirectories
- **[Model](./Model)**: Models for Facebook graph responses.
