# Ophelia Integration Google Login

Standard OAuth2 implementation for Google authentication.

## 📂 Core Services

### [GoogleLoginService.cs](./GoogleLoginService.cs)
Facilitates the authentication flow with Google APIs to retrieve user profiles.
- **Key Methods**: `GetUserDataAsync`, `ValidateToken`.

## 📁 Subdirectories
- **[Model](./Model)**: Models for Google User Profile and Token info.
