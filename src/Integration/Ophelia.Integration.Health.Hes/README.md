# Ophelia Integration Health Hes

Integration with the Turkish Ministry of Health HES (Hayat Eve Sığar) status verification services.

## 📂 Core Services

### [HesService.cs](./HesService.cs)
Provides methods to verify HES codes, check risk status, and interface with the official API endpoints.
- **Key Methods**: `VerifyCode`, `GetRiskStatus`.

## 📁 Subdirectories
- **[Model](./Model)**: Request and response objects for the HES service API.
