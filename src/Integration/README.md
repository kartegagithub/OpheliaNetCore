# Ophelia Integration

This directory contains various integration modules that allow the Ophelia Framework to communicate with third-party services, APIs, and platforms.

## 📂 Available Integrations

| Integration | Description |
| :--- | :--- |
| **Amazon** | Connectivity for AWS services like S3. |
| **Apple/Google/Facebook Login** | OAuth implementations for social authentication. |
| **CDN (Vimeo)** | Integration with video hosting and delivery services. |
| **Documents** | Processing and managing external document formats. |
| **GitHub** | Integration with GitHub API for repository and issue management. |
| **Health (Hes)** | Integration with specific healthcare/HES code services. |
| **I18NService** | External Internationalization and translation services. |
| **LDAP** | Directory service integration for user authentication. |
| **Notification** | Hub for sending SMS, Push, and Email notifications. |
| **Redis** | Integration for high-performance distributed caching and messaging. |
| **StockExchange** | Financial data integration for stock market services. |
| **Weather** | Real-time weather data retrieval. |

---
*Each subdirectory contains a specific project with its own dependencies and configuration.*
