# Ophelia Integration Stock Exchange

A service for retrieving real-time and historical financial data from global and local stock exchanges.

## 📂 Core Services

### [StockService.cs](./StockService.cs)
Provides high-level methods to fetch stock prices, market trends, and company financials.
- **Key Methods**: `GetPriceAsync`, `GetMarketCap`, `GetClosingPrices`.

## 📁 Subdirectories
- **[Model](./Model)**: Models for financial instruments, stock quotes, and market entities.
