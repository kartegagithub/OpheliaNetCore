# Ophelia Core Unit Tests

Unit tests covering the fundamental logic of the Ophelia framework.

## 📂 Test Areas

- **[Extensions](./Extensions)**: Verifies string, date, and collection extension methods.
- **[Cryptography](./Cryptography)**: Ensures encryption and hashing algorithms are working as expected.
- **[Reflection](./Reflection)**: Tests for dynamic object access and property mapping.
- **[Net](./Net)**: Tests for HTTP request factory and networking utilities.
- **[Tasks](./Tasks)**: Validates background job management and scheduling logic.
